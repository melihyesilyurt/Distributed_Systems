Yazdığım kodların fotoğrafları "files.png" içerisindedir.

nslookup ftp, nslookup www, nslookup mail komutlarının sonuçları "nslookup.png" içerisindedir.

ping melih.local sonucu "ping.png" içerisindedir.

nslookup 192.168.77.2 "nslookup ip.png" içerisindedir.
