Bittorent sistemini kullanan bir server oluşturdum bu server sayesinde istediğiniz filmi aratabilirsiniz.
Seçtiğiniz filmi ayarladğınız görüntü kalitesine göre isterseniz indirebilir veya VLC üzerinden oynatabilirsiniz.

Çalıştırmak için aşağıdaki kurulumları gerçekleştirmeniz gerekmektedir ve bilgisayarınızda VLC yüklü olmak zorundadır.

pip install requests
pip install tabulate
npm install webtorrent-cli -g

Uygulamayı başlatmak için terminale python index.py yazmanız gerekmektedir.

Uygulamanın çalışmasıyla alakalı fotoğraflar sırasıyla "1.png", "2.png", "3.png" olarak dosya dizinine eklenmiştir.
