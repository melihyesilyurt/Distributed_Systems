How is the Project Run?
1-) run server.java
2-) run client.java
3-) To use the project, simply follow the instructions in the terminal.

Description of the project:
The project is a car sales program. The user can add a car and sales receipt to the database used on the server by sending a request to the server through the client.
User can also search by Sales receipt, Car make and car serial number by Seller name and buyer ID number.
In order to search, the information requested to be searched is sent to the server via the client and this information is searched in the database by the server. The result is sent back to the client.
